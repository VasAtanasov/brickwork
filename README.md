# Brickwork 

#### Technologies

- Java 11
- Maven
- Junit 5

##### Package and run the tests

```shell script
mvn clean pacakte
```

##### Run the application:

```shell script
java -cp target\brickwork-1.0-SNAPSHOT.jar com.github.vasatanasov.brickwork.App
```
